# H2 CRM PWA Wrapper

## Deploy to GitHub Pages (free)

1. Create a new repo at github.com (e.g. `h2crm-app`) — set to **Public**
2. Upload all 5 files in this folder
3. Go to repo Settings → Pages → Source: **main branch / root**
4. Your PWA URL: `https://[your-username].github.io/h2crm-app`

## Before uploading

Open `index.html` and replace:
  YOUR_GAS_WEBAPP_URL_HERE
with your actual GAS Web App URL.

## Install on Android

1. Open Chrome → go to your GitHub Pages URL
2. Chrome shows **"Add to Home screen"** banner automatically
3. Tap Install → app appears on home screen with H2 Patios icon
4. Opens fullscreen, no browser bar

## Install on iPhone

1. Open Safari → go to your GitHub Pages URL  
2. Tap Share button → **"Add to Home Screen"**
3. App appears with H2 Patios icon
